# LinkDict

基于有道在线 API 的 Obsidian 词典插件。100% 由 Vibe Coding 完成。

## 功能

- **在线查词**：使用有道在线 API 查询单词并生成包含音标、释义、变形的 Markdown 笔记。
- **Lemma 识别**：查询变形词时自动识别并定位至原形。
- **双链替换**：支持在查询时将选中的单词自动替换为指向原形的双向链接。
- **悬浮查词**：选中文本后通过右键菜单快速查看单词释义。
- **侧边栏视图**：提供独立的侧边栏词典视图，方便持续查词。
- **悬浮窗**：提供选词查询的悬浮窗。

## 安装

1. 使用 BRAT 插件添加本仓库地址进行安装。
2. 或在 Releases 页面手动下载所有文件放入插件目录。

## 版本说明

- **v2.0.0+**: 使用有道在线 API，需要网络连接，数据更全面、更新及时。
- **v1.0.3**: 旧版本，基于 ECDICT 本地数据，完全离线使用。
