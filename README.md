# LinkDict

基于 ECDICT 数据的 Obsidian 离线词典插件。100% 由 Vibe Coding 完成。

## 功能

- **本地查词**：查询单词并生成包含音标、释义、变形的 Markdown 笔记。
- **Lemma 识别**：查询变形词时自动识别并定位至原形。
- **双链替换**：支持在查询时将选中出的单词自动替换为指向原形的双向链接。

## 安装

1. 使用 BRAT 插件添加本仓库地址进行安装。dictionary.json 文件需要手动导入(已在设置增加中手动下载)。
2. 或在 Releases 页面手动下载所有文件放入插件目录。
